## - Sorting Algorithm
      -  Selection Sort
      step 1 Set minimum value to array[0] aka index = position  | > 0 < | 1 | 2| ....| >
      step 2 search the minimum element in the array
      step 3 swap with value in min variable
      step 4 increment min variable to point to next level
